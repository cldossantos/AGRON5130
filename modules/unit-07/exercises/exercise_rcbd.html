ANOVA

For this analysis of variance, we need two lines of code. The first is the model statement:

barley_model = aov(yield ~ block + gen, data=barley)

In the above code:

  • barley_model defines the object to which the ANOVA results will be saved
  • â€œaovâ€ tells R to run an analysis of variance
  • â€œyield ~ block + genâ€ tells R our linear additive model. When using R, we leave out mu and the error term â€“ R knows to calculate these.
  • â€œdata=barleyâ€ tells R to fit our ANOVA model to the barley dataset

Reviewing our results is simple:

Plotting the Results

Although we get into the presentation of treatment means in more detail late in the course, lets take a quick look at our treatment means. We will use ggplot.

barley_plot = ggplot(barley, aes(x=gen, y=yield)) +
  geom_bar(stat = "summary", fun="mean")

In the plot above: * barley is the object to which we are saving the plot * ggplot() starts the ggplot function * â€œggplot(barley, aes(x=gen, y=yield))â€ tells R that our plot will be based on the barley dataset. Anything in ggplot that occurs in aes() brackets is an â€œaestheticâ€. An aesthetic is any value that is based on a variable from our plot and changes in value from treatment to treatment. x=gen tells R the horizontal position of whatever we draw (points, parts, etc.) is determined by the gen variable. y=yield tells R the vertical position (height) of our object is based on the yield variable. * â€œgeom_barâ€ tells R we want to draw bars with our data. * With geom bar, we can also calculate summary statistics on the fly. stat=â€œsummaryâ€ tells R we are going to calculate a summary statistic. fun=â€œmeanâ€ tells R to use the mean as that statistic.

Now we can display the plot at any time by running barley_plot:
