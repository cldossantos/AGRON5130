# AGRON 5130 - Data Science for Agricultural Professionals


# Course Description

Quantitative methods for analyzing and
interpreting agronomic information. Principles of
experimental design, hypothesis testing, analysis
of variance, regression, correlation, and
graphical representation of data. Use of R and R
Studio for organizing, analyzing, and presenting
data. Required course for the Master of Science in
Agronomy degree program.
